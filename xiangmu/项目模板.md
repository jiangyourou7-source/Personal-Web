# 新增项目模板

将此文档填写完成后发给我，我会将新项目添加到网站的"我的项目"页面中。

---

## 项目信息

| 字段 | 说明 | 示例 |
|------|------|------|
| 项目名称 | 项目的简短名称 | AI 写作助手 |
| 项目描述 | 一句话描述项目是做什么的 | 基于大语言模型的写作辅助工具，支持文章生成和内容优化 |
| 项目状态 | 计划中 / 进行中 / 已完成 / 已暂停 | 进行中 |
| 进度百分比 | 0-100 的数字 | 65 |
| 项目分类 | Web应用 / 移动应用 / 工具 / AI应用 / 其他 | AI应用 |
| 技术标签 | 用逗号分隔 | Next.js, OpenAI API, Prisma |
| 开始时间 | 格式 YYYY-MM | 2026-04 |
| 目标完成时间 | 格式 YYYY-MM（可选） | 2026-06 |
| 实际完成时间 | 格式 YYYY-MM（已完成时填写） | - |
| 项目链接 | 线上地址（可选） | https://example.com |

## 功能特性

列出项目的主要功能，每行一个：

- 功能一
- 功能二
- 功能三

## 最新动态

记录项目的关键进展，格式：日期 + 内容

| 日期 | 内容 |
|------|------|
| YYYY-MM-DD | 做了什么事情 |
| YYYY-MM-DD | 做了什么事情 |
